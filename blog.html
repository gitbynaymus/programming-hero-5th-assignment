Question-1: What are the different ways to select an element in the DOM?

Answer:

There are many ways to select an element in the Dom

  • getElementById()
  • getElementsByClassName()
  • getElementsByTagName()
  • querySelector()
  • getElementsByName()

Question-2: What are the different between innerHTML, innerText, and textContent?

Answer:

  • innerHTML: innerHTML is a property in JavaScript that allows to get or set the HTML content within an element.
  • innerText: innerText retrieves or sets the text content within an HTML element
  • textContent: The textContent property sets or returns the text content of the specified node, and all its descendants.

Question-3: What is event delegation in the DOM?

Answer:

Event delegation refers to the process of using event propagation (bubbling) to handle events at a higher level in the DOM than the element on which the event originated. It allows us to attach a single event listener for elements that exist now or in the future.

Question-4: What is event bubbling in the DOM?

Answer:

Event bubbling in JavaScript is a mechanism where an event triggered on a child element propagates upward through its ancestors in the DOM. It allows parent elements to respond to events triggered by their child elements.

Question-5: How do you create, add, and remove elements using JavaScript?

Answer:

  • createElement(): Creates a new element.
  • appendChild(): Adds the element to the end of a parent.
  • insertBefore(): Adds the element before a specified child.
  • remove(): Removes the element directly.
  • removeChild(): Removes a child element from its parent.
